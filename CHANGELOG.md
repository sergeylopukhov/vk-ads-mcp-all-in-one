# Changelog

Все заметные изменения фиксируются в этом файле.

## 1.2.5 — 2026-07-22

- Добавлен read-only инструмент `vk_rescore_community_research_run` для пересчёта завершённого сохранённого исследования по новому профилю без поиска, чтения публикаций и запросов к VK.
- Пересчёт создаёт новый снимок с новым `run_id`, сохраняет ссылку на исходный запуск и не меняет рекламный кабинет.

## 1.2.4 — 2026-07-22

- Добавлен универсальный профиль скоринга `community-research-v2`: нормированные совпадения в названии, описании и постах, свежесть, доля тематических публикаций, штраф низкой активности и исключающие термины.
- Неполные `scoring_rules` теперь переопределяют только явно указанные параметры и не отключают остальные базовые сигналы. Вес `0` отключает сигнал намеренно.
- В снимке исследования сохраняется версия формулы, чтобы результаты разных запусков можно было отличить.

## 1.2.3 — 2026-07-22

- Фоновое исследование сообществ отправляет безопасный статус через стандартные MCP-уведомления раз в минуту и отдельное уведомление по завершении.
- В уведомлениях есть только счётчики обработки; токены, тексты постов и данные рекламного кабинета не передаются.
- Добавлен тест доставки прогресса и итога через MCP; тест write-ошибки больше не зависит от внешнего соединения.

## 1.2.0 — 2026-07-22

- Добавлены read-only инструменты поиска, анализа, скоринга и экспорта публичных сообществ VK.
- Для сообществ поддерживаются legacy OAuth и VK ID OAuth; установщик предлагает встроенное приложение и позволяет указать своё.
- Исправлены актуальный callback `oauth.vk.ru`, разбор `groups.getById` и разделение rate limit между Core VK API и VK Ads.
- Экспорт кандидатов остаётся локальным и помечает каждую строку `pending_approval`.
- `vk_score_communities` строго проверяет `scoring_rules` и объясняет ошибки вместо молчаливого нулевого результата.

## 0.1.0 — 2026-07-20

- Первый публичный выпуск локального MCP-сервера для VK Ads API.
- Безопасный режим чтения, профили, secret store, preview и подтверждения записи.
- Инструменты управления рекламой, аналитики, медиа, аудиторий, отчётов и экспорта.
